## 涉及技术：

### SpringBoot+Mybatis+Thymeleaf+Ehcache+Layui

+ 后端框架： SpringBoot
+ 持久化层框架： Mybatis
+ 模版引擎： Thymeleaf
+ 缓存： Ehcache
+ 前端引擎： Layui
